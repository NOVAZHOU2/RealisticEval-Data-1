```java
/**
 * Gets the line number in the content at the specified index.
 *
 * @param content - The string content to check.
 * @param index - The character index to find the line number for.
 * @return The line number corresponding to the given index.
 */
public int getLineNumber(String content, int index) {
    if (content == null || content.isEmpty()) {
        return 0;
    }

    int lineNumber = 1; // Start counting from line 1

    for (int i = 0; i < index; i++) {
        if (content.charAt(i) == '\n') {
            lineNumber++;
        }
    }

    return lineNumber;
}
```

### Explanation:
- **Parameters**: 
  - `content`: A `String` representing the text content.
  - `index`: An `int` representing the character index.

- **Return Value**:
  - Returns an `int` which is the line number corresponding to the given index.

- **Logic**:
  - Initialize `lineNumber` to 1 because line numbering typically starts from 1.
  - Iterate through each character of the content up to the specified index.
  - If the current character is a newline (`'\n'`), increment the `lineNumber`.

This function will correctly calculate the line number based on the provided character index in the content string.
