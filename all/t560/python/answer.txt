```python
def get_line_number(content: str, index: int) -> int:
    """
    Gets the line number in the content at the specified index.

    :param content: The string content to check.
    :param index: The character index to find the line number for.
    :return: The line number corresponding to the given index.
    """
    lines = content.split('\n')
    cumulative_length = 0

    for i, line in enumerate(lines):
        if cumulative_length + len(line) >= index:
            return i + 1
        cumulative_length += len(line) + 1  # Add 1 for the newline character

    return len(lines)  # If index is beyond the last character of the last line
```

This Python function `get_line_number` takes two parameters: `content` (a string) and `index` (an integer). It splits the content into lines and iterates through them to find the line number corresponding to the specified index. If the index is beyond the last character of the last line, it returns the total number of lines.
