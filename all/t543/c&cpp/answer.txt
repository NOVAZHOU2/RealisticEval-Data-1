```cpp
#include <iostream>

/**
 * Increments the entered number.
 * If the number is non-positive (<= 0), returns the original value.
 * If the number is positive, returns the value plus 1.
 *
 * @param num - The number to increment.
 * @return The incremented value or the original number.
 */
int incrementNumber(int num) {
    if (num <= 0) {
        return num;
    } else {
        return num + 1;
    }
}

// Example usage
int main() {
    int num = 5;
    std::cout << "Original number: " << num << std::endl;
    std::cout << "Incremented number: " << incrementNumber(num) << std::endl;

    num = -3;
    std::cout << "Original number: " << num << std::endl;
    std::cout << "Incremented number: " << incrementNumber(num) << std::endl;

    return 0;
}
```

- **Function Definition**: The function `incrementNumber` takes an integer `num` as input and returns an integer.
- **Condition Check**: 
  - If `num` is less than or equal to zero, it returns `num`.
  - Otherwise, it returns `num + 1`.
- **Main Function**: Demonstrates the usage of `incrementNumber` with example inputs.
