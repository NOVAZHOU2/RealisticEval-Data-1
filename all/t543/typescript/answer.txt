Certainly! Below is the TypeScript version of your JavaScript function:

```typescript
/**
 * Increments the entered number.
 * If the number is non-positive (<= 0), returns the original value.
 * If the number is positive, returns the value plus 1.
 *
 * @param {number} num - The number to increment.
 * @returns {number} - The incremented value or the original number.
 */
function incrementNumber(num: number): number {
    if (num <= 0) {
        return num;
    } else {
        return num + 1;
    }
}
```

In this TypeScript version:
- I added type annotations for the input parameter `num` and the return value, both of which are `number`.
- The function body remains the same as in the original JavaScript code.
