Certainly! Here's the JavaScript function converted to TypeScript:

```typescript
/**
 * Increments the entered number.
 * If the number is non-positive (<= 0), returns the original value.
 * If the number is positive, returns the value plus 1.
 *
 * @param {number} num - The number to increment.
 * @returns {number} - The incremented value or the original number.
 */
function incrementNumber(num: number): number {
    if (num <= 0) {
        return num;
    } else {
        return num + 1;
    }
}
```

In this TypeScript version, I've added type annotations for the `num` parameter and the return type of the function. This helps with static type checking and makes it clear what types of values the function expects and returns.
