```java
/**
 * Increments the entered number.
 * If the number is non-positive (<= 0), returns the original value.
 * If the number is positive, returns the value plus 1.
 *
 * @param num - The number to increment.
 * @return The incremented value or the original number.
 */
public int incrementNumber(int num) {
    if (num <= 0) {
        return num;
    } else {
        return num + 1;
    }
}
```

In this Java version:
- The method `incrementNumber` takes an integer parameter `num`.
- It checks if the number is non-positive (`<= 0`). If true, it returns the original number.
- If the number is positive, it returns the number incremented by 1.
