Here's the Python code:

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

def remove_ads():
    # Initialize the WebDriver (replace 'path/to/chromedriver' with the path to your ChromeDriver executable)
    driver = webdriver.Chrome(executable_path='path/to/chromedriver')
    
    # Open the web page
    driver.get('http://example.com')  # Replace with the actual URL of the web page
    
    # Find all elements with the specific class name and remove them
    ad_class_name = 'ad-class-name'  # Replace with the actual class name of the ads
    ads_to_remove = driver.find_elements(By.CLASS_NAME, ad_class_name)
    
    for ad in ads_to_remove:
        ad.click()  # Assuming clicking the ad will remove it
        # Alternatively, you can use JavaScript to remove the element
        # driver.execute_script("arguments[0].parentNode.removeChild(arguments[0]);", ad)
    
    # Optionally, you can save the modified page or take a screenshot
    # driver.save_screenshot('modified_page.png')
    
    # Close the browser
    driver.quit()

# Call the function to remove ads
remove_ads()
```

### Explanation:
1. **WebDriver Initialization**: The script initializes a Chrome WebDriver. Make sure to replace `'path/to/chromedriver'` with the actual path to your ChromeDriver executable.
2. **Open Web Page**: The script opens the specified web page using `driver.get()`.
3. **Find and Remove Ads**: It finds all elements with the specified class name using `find_elements()` and removes them by clicking on them. Alternatively, you can use JavaScript to remove the elements directly.
4. **Close Browser**: Finally, the script closes the browser using `driver.quit()`.

Make sure to replace `'http://example.com'` and `'ad-class-name'` with the actual URL of the web page and the class name of the ads you want to remove.
