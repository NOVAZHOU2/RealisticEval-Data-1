Sure! Here is the equivalent Java code using Selenium WebDriver to iterate over list items on a webpage and remove ads that contain a specific class name:

```java
import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;

import java.util.List;

public class RemoveAds {
    public static void main(String[] args) {
        // Set up ChromeDriver
        System.setProperty("webdriver.chrome.driver", "path/to/chromedriver");
        WebDriver driver = new ChromeDriver();

        // Open the web page
        driver.get("http://example.com");

        // Find all list items on the page
        List<WebElement> listItems = driver.findElements(By.tagName("li"));

        // Loop through each list item
        for (WebElement listItem : listItems) {
            // Check if the list item contains the specific class name
            if (listItem.getAttribute("class").contains("ad-class")) {
                // Remove the ad element from the DOM
                listItem.remove();
            }
        }

        // Close the browser
        driver.quit();
    }
}
```

### Explanation:
1. **Set Up ChromeDriver**: The `System.setProperty` method sets the path to the ChromeDriver executable.
2. **Open the Web Page**: The `driver.get` method opens the specified URL in the browser.
3. **Find All List Items**: The `findElements` method finds all elements with the tag name "li".
4. **Loop Through Each List Item**: A loop iterates through each list item.
5. **Check Class Name**: The `getAttribute("class")` method retrieves the class attribute of the list item, and the `contains` method checks if it contains the specific class name ("ad-class").
6. **Remove Ad Element**: If the condition is met, the `remove` method removes the ad element from the DOM.
7. **Close the Browser**: Finally, the `quit` method closes the browser.

Make sure to replace `"path/to/chromedriver"` with the actual path to your ChromeDriver executable and `"http://example.com"` with the URL of the webpage you want to interact with. Also, replace `"ad-class"` with the actual class name of the ads you want to remove.
