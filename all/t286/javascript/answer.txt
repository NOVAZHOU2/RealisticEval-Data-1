Here is the equivalent JavaScript function for your Python function:

```javascript
function findLargestDivisible(n) {
    let halfN = Math.floor(n / 2);

    for(let i = n; i >= halfN; i--) {
        if(i % 10 === 0 || i % 5 === 0) {
            return i;
        }
    }

    return null;
}
```

This function works similarly to your Python function. It iterates from `n` down to `halfN`, checking each number to see if it's divisible by 10 or 5. If it finds a number that fits these criteria, it returns that number. If it goes through all numbers in the range without finding a match, it returns `null`.
