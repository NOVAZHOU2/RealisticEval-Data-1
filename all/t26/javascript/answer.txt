Here is the JavaScript equivalent of your Python function:

```javascript
function convertToCommaSeparated(inputString) {
    /**
     * Convert input strings that contain multiple separators (for example, * ; /) 
     * to comma-separated formatted strings
     *
     * @param {string} inputString - The input string containing various separators 
     *                               like *, ;, /, -, :
     * @returns {string} - A comma-separated string where all specified separators 
     *                    have been replaced with commas.
     */
    return inputString.replace(/[*;\/\-:]/g, ',');
}
```

This JavaScript function uses the `replace()` method in combination with a regular expression (`/[*;\/\-:]/g`) to replace all occurrences of the specified separators with commas. The `g` flag at the end of the regular expression indicates a global search (i.e., it will replace all instances rather than just the first one).
