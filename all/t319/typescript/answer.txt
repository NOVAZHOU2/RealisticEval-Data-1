```typescript
/**
 * Count the number of dashes in a string
 *
 * @param {string} str - The string from which to count dash characters.
 * @returns {number} - The total count of dash characters found in the string.
 */
function countDashes(str: string): number {
    return (str.match(/-/g) || []).length;
}
```

In this TypeScript version:
- The `countDashes` function now explicitly takes a parameter `str` of type `string`.
- The return type of the function is specified as `number`.
- The regular expression `/-/g` is used to match all occurrences of dashes in the string. If no matches are found, `match` returns `null`, so we use `(str.match(/-/g) || [])` to ensure we have an array to work with, and then get its length using `.length`.
