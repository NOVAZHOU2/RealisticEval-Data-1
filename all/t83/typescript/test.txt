Here's how you can convert your Python unittest test class to TypeScript using Jest:

```typescript
import { describe, it, expect } from '@jest/globals';

describe('rotateListElements', () => {
  it('should rotate the elements of the list to the left by one position', () => {
    const input = [1, 2, 3, 4, 5];
    const expectedOutput = [2, 3, 4, 5, 1];

    const result = rotateListElements(input);

    expect(result).toEqual(expectedOutput);
  });

  // Add more tests here...
});
```

Please note that you need to have Jest installed in your project to run these tests. If not already installed, you can do so by running `npm install --save-dev jest`. Also, make sure to add `"test": "jest"` script in your package.json file to run tests using `npm test` command.
