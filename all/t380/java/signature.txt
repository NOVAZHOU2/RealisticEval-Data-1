Here is how you can convert your Python code signature to Java code signature while keeping the same comments:

```java
/**
 * Calculate the total number of seconds given an array of time periods in the order of
 * days, hours, minutes, and seconds.
 *
 * @param time An array of integers, where
 *             time[0] - number of days (optional)
 *             time[1] - number of hours (optional)
 *             time[2] - number of minutes (optional)
 *             time[3] - number of seconds (optional)
 * @return The total number of seconds
 *
 * Examples:
 *     calculateTotalSeconds(new int[]{1, 2, 3, 4}) returns 93784
 *     calculateTotalSeconds(new int[]{0, 2, 3}) returns 7380
 */
public static int calculateTotalSeconds(int[] time) {
    // Method implementation goes here...
}
```

In Java, we use `int[]` instead of `Tuple[int]`. Also, Java doesn't support optional parameters like Python does. Therefore, all parameters are mandatory in Java. If some parameters are optional in Python, they should be checked for null values inside the method body before using them.
