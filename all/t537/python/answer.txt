```python
from datetime import datetime

def get_time():
    """
    Gets the current time formatted as 'hh:mm AM/PM'.

    :return: The formatted time string.
    :rtype: str
    """
    now = datetime.now()
    return now.strftime("%I:%M %p")

# Example usage:
formatted_time = get_time()
print(formatted_time)
```

This Python function uses the `datetime` module to get the current time and format it as 'hh:mm AM/PM'.
