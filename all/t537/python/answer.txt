Sure! Here is the equivalent Python function that gets the current time formatted as 'hh:mm AM/PM':

```python
from datetime import datetime

def get_time():
    """
    Gets the current time formatted as 'hh:mm AM/PM'.

    :return: The formatted time string.
    :rtype: str
    """
    now = datetime.now()
    return now.strftime("%I:%M %p")

# Example usage:
print(get_time())
```

This Python function uses the `datetime` module to get the current time and then formats it using the `strftime` method with the format specifier `%I:%M %p`, which corresponds to 'hh:mm AM/PM'.
