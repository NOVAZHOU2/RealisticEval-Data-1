```python
def filter_array(unfiltered_array, is_qualified):
    """
    Filters elements from an array based on a qualification function.

    :param unfiltered_array: The array to filter.
    :param is_qualified: The function that determines if an element qualifies.
    :return: A new list containing the elements that qualify.
    """
    return [element for element in unfiltered_array if is_qualified(element)]
```

This Python function `filter_array` takes two parameters:
- `unfiltered_array`: The list of elements to be filtered.
- `is_qualified`: A function that takes an element and returns `True` if the element qualifies, otherwise `False`.

The function uses a list comprehension to create a new list containing only the elements for which `is_qualified` returns `True`.
