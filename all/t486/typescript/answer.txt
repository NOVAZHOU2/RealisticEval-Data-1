Here's how you can convert your Python code to TypeScript:

```typescript
class Calculator {
    add(a: number, b: number): number {
        /**
         * Returns the sum of a and b.
         */
        return a + b;
    }

    subtract(a: number, b: number): number {
        /**
         * Returns the difference of a and b.
         */
        return a - b;
    }

    multiply(a: number, b: number): number {
        /**
         * Returns the product of a and b.
         */
        return a * b;
    }

    divide(a: number, b: number): number {
        /**
         * Returns the quotient of a and b.
         * Throws Error if b is zero.
         */
        if (b === 0) {
            throw new Error("Cannot divide by zero.");
        }
        return a / b;
    }
}
```

In TypeScript, we use `number` instead of `float`, and we use `throw new Error()` instead of `raise ValueError()`. Also note that in TypeScript, all methods must have a return type specified, even if it's `void`. In this case, I've assumed that each method should return a `number`, but you might want to adjust this based on your actual implementation.
