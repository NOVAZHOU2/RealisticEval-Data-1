Here's the equivalent Java code:

```java
import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class Answer {
    
    public static Object cleanPattern(String x, String pattern) {
        Pattern p = Pattern.compile(pattern);
        Matcher m = p.matcher(x);

        if(m.find()){
            try{
                return Float.parseFloat(m.group());
            } catch(NumberFormatException e){
                return "";
            }
        }

        return "";
    }
}
```

Please note that in Java, `Object` is used instead of `Union[str, float]` because Java doesn't have a built-in union type like Python does. Also, error handling is done using exceptions and the method returns either a `String` or a `Float`.
