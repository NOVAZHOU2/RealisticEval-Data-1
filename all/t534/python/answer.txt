```python
def remove_element_in_array(array, element):
    """
    Removes the first occurrence of a specified element from an array.

    :param array: The array from which to remove the element.
    :param element: The element to remove from the array.
    :return: A new list with the element removed, or the original list if the element is not found.
    """
    try:
        index = array.index(element)
        return array[:index] + array[index+1:]
    except ValueError:
        return array

# Example usage:
my_array = [1, 2, 3, 4, 5]
element_to_remove = 3
new_array = remove_element_in_array(my_array, element_to_remove)
print(new_array)  # Output: [1, 2, 4, 5]
```

### Explanation:
- **`array.index(element)`**: This method returns the index of the first occurrence of `element` in `array`. If `element` is not found, it raises a `ValueError`.
- **`array[:index] + array[index+1:]`**: This creates a new list that excludes the element at the found index.
- **`try...except ValueError`**: This block handles the case where the element is not found in the array, returning the original array unchanged.
