Here is the JavaScript equivalent of your Python function:

```javascript
function safeFormat(template, ...args) {
    // Convert args into an object for easier access
    let obj = {};
    for(let i=0; i<args.length; i++) {
        if(typeof args[i] === 'object' && args[i] !== null){
            Object.assign(obj, args[i]);
        }
    }

    return template.replace(/{(\w+)}/g, function(match, key) {
        return key in obj ? obj[key] : match;
    });
}
```

In this JavaScript function, we're using the `replace` method along with a regular expression to find all placeholders `{key}` in the `template`. For each placeholder, we check if there's a corresponding value in the `obj` (our JavaScript version of `kwargs`). If there is, we replace the placeholder with its value, otherwise, we keep the original placeholder. 

Please note that this JavaScript function can accept multiple objects as argument and merge them into one object before formatting the template. This is done because JavaScript doesn't support variable-length keyword arguments like Python does (`**kwargs`).
