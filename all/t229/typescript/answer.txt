Here is the equivalent TypeScript code for your Python function:

```typescript
function convertFileSize(sizeBytes: number): string {
    /**
     * Converts a file size in bytes to a human-readable format.
     * For example:
     *      input: 2120
     *      output: 2KB
     * Args:
     *      sizeBytes (number): The size in bytes to be converted.
     *
     * Returns:
     *      string: The converted size in a human-readable format (e.g., "2KB", "1MB").
     */

    const units = ['B', 'KB', 'MB', 'GB', 'TB', 'PB'];
    let i = Math.floor(Math.log2(sizeBytes) / 10);
    return `${(sizeBytes / Math.pow(1024, i)).toFixed(2)}${units[i]}`;
}
```

This TypeScript function `convertFileSize` does the same thing as your Python function - it converts a given size in bytes to a more readable format like KB, MB, etc. It uses the logarithm base 2 to determine the appropriate unit and then divides the original size by the corresponding power of 1024 to get the value in that unit. The result is then formatted to two decimal places and appended with the unit symbol.
