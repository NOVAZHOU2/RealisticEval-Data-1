Here is the JavaScript equivalent of your Python function:

```javascript
/**
 * Converts a file size in bytes to a human-readable format.
 * For example:
 *      input: 2120
 *      output: 2KB
 * @param {number} sizeBytes - The size in bytes to be converted.
 * @returns {string} The converted size in a human-readable format (e.g., "2KB", "1MB").
 */
function convertFileSize(sizeBytes) {
    let sizes = ['B', 'KB', 'MB', 'GB', 'TB'];
    if (sizeBytes == 0) return '0B';
    let i = parseInt(Math.floor(Math.log(sizeBytes) / Math.log(1024)));
    return (sizeBytes / Math.pow(1024, i)).toFixed(2) + ' ' + sizes[i];
}
```

This JavaScript function does exactly what your Python function does, it converts a file size from bytes into a more readable format like KB, MB, GB etc. It also includes a comment at the top explaining its purpose and usage.
