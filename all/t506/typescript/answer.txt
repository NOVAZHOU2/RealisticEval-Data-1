```typescript
function snakeToCamel(snakeStr: string): string {
    /**
     * Convert a snake_case string to CamelCase.
     *
     * @param {string} snakeStr - The snake_case string to convert.
     * @returns {string} - The converted CamelCase string.
     */
    
    return snakeStr.replace(/_([a-z])/g, (match, letter) => letter.toUpperCase());
}
```

This TypeScript function uses a regular expression to find all instances of an underscore followed by a lowercase letter and replaces them with the uppercase version of the letter. This effectively converts the snake_case string to CamelCase.
