Here's the equivalent JavaScript function for your Python code:

```javascript
function snakeToCamel(snakeStr) {
    /**
     * Convert a snake_case string to CamelCase.
     *
     * @param {string} snakeStr - The snake_case string to convert.
     * @return {string} The converted CamelCase string.
     */
    return snakeStr.replace(/(_\w)/g, function(match) {
        return match[1].toUpperCase();
    });
}
```

This JavaScript function uses regular expressions and the `replace` method to iterate over all occurrences of an underscore followed by a character (`_\w`). For each occurrence, it replaces it with the uppercase version of the character that follows the underscore.
