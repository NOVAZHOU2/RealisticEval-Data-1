```javascript
/**
 * Convert a snake_case string to CamelCase.
 *
 * @param {string} snakeStr - The snake_case string to convert.
 * @returns {string} - The converted CamelCase string.
 */
function snakeToCamel(snakeStr) {
    // Split the string by underscore and map each word to capitalize it
    return snakeStr.split('_').map(word => 
        word.charAt(0).toUpperCase() + word.slice(1)
    ).join('');
}
```

This JavaScript function `snakeToCamel` does the same thing as your Python function `snake_to_camel`. It splits the input string by underscores, capitalizes the first letter of each word, and then joins them back together without any separators.
