```java
package org.real.temp;

import java.util.ArrayList;
import java.util.HashSet;
import java.util.List;
import java.util.Set;

public class Answer {

    public static List<String> findDuplicateIps(List<String> ipList, List<String> ignoreList) {
        Set<String> uniqueIpSet = new HashSet<>();
        Set<String> duplicateIpSet = new HashSet<>();

        for(String ip : ipList){
            if(!ignoreList.contains(ip)){
                if(uniqueIpSet.contains(ip)){
                    duplicateIpSet.add(ip);
                } else {
                    uniqueIpSet.add(ip);
                }
            }
        }

        return new ArrayList<>(duplicateIpSet);
    }
}
```

In this Java version, we're using `HashSet` to keep track of seen IPs and duplicates, and `ArrayList` to return the result. The method iterates over each IP address in the `ipList`. If it's not in the `ignoreList`, it checks whether it has been encountered before. If yes, it adds it to the `duplicateIpSet`; otherwise, it adds it to the `uniqueIpSet`. Finally, it returns the `duplicateIpSet` converted to an `ArrayList`.
