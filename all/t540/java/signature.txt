```java
import java.util.ArrayList;
import java.util.List;

/**
 * Generates all unique combinations of pairs from an array.
 *
 * @param array - The input array from which combinations are generated.
 * @return A list of lists, where each inner list contains a unique pair of elements.
 */
public List<List<Integer>> generateUniquePairs(int[] array) {
    List<List<Integer>> result = new ArrayList<>();
    for (int i = 0; i < array.length; i++) {
        for (int j = i + 1; j < array.length; j++) {
            List<Integer> pair = new ArrayList<>();
            pair.add(array[i]);
            pair.add(array[j]);
            result.add(pair);
        }
    }
    return result;
}
```

This Java method `generateUniquePairs` takes an integer array as input and returns a list of lists containing unique pairs of elements from the input array.
