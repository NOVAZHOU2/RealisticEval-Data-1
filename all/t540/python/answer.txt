```python
from itertools import combinations

def generate_unique_pairs(array):
    """
    Generates all unique combinations of pairs from an array.

    :param array: The input array from which combinations are generated.
    :return: A list of lists, where each inner list contains a unique pair of elements.
    """
    return list(combinations(array, 2))

# Example usage:
array = [1, 2, 3, 4]
pairs = generate_unique_pairs(array)
print(pairs)
```

In this Python code, we use the `combinations` function from the `itertools` module to generate all possible unique pairs from the input array. The `combinations` function takes two arguments: the iterable (in this case, the array) and the length of the combinations (which is 2 for pairs). The result is then converted to a list and returned.
