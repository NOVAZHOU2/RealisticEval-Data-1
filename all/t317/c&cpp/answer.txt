```cpp
#include <iostream>
#include <string>

/**
 * Count the number of letters in a string.
 *
 * @param str - The input string from which to count letters.
 * @return The count of letters in the string.
 */
int countLetters(const std::string& str) {
    int letterCount = 0;
    for (char ch : str) {
        if ((ch >= 'a' && ch <= 'z') || (ch >= 'A' && ch <= 'Z')) {
            ++letterCount;
        }
    }
    return letterCount;
}

// Example usage
int main() {
    std::string input = "Hello, World!";
    int result = countLetters(input);
    std::cout << "Number of letters: " << result << std::endl; // Output: Number of letters: 10
    return 0;
}
```

This C++ function `countLetters` takes a constant reference to a `std::string` and counts the number of alphabetic characters (both uppercase and lowercase). It iterates through each character in the string and increments the `letterCount` if the character is an alphabet. The example usage demonstrates how to call the function and print the result.
