Here's how you can convert your Python unittest test class to a TypeScript test case using Jest:

```typescript
import { describe, it, expect } from '@jest/globals';
import { findNthWeekdayOfSpecificYear } from './yourFile'; // Adjust the path accordingly

describe('findNthWeekdayOfSpecificYear', () => {
    it('should return the correct date for the nth weekday of a specific year, month, and weekday', () => {
        const result = findNthWeekdayOfSpecificYear(2023, 4, 1, 0); // Example parameters
        expect(result).toBeInstanceOf(Date);
        expect(result.getFullYear()).toBe(2023);
        expect(result.getMonth() + 1).toBe(4); // Note: getMonth() returns 0-based index
        expect(result.getDay()).toBe(0); // Sunday
    });

    it('should return the last occurrence of the weekday if the nth occurrence does not exist', () => {
        const result = findNthWeekdayOfSpecificYear(2023, 2, 5, 0); // February 2023 has only 4 Sundays
        expect(result).toBeInstanceOf(Date);
        expect(result.getFullYear()).toBe(2023);
        expect(result.getMonth() + 1).toBe(2); // Note: getMonth() returns 0-based index
        expect(result.getDay()).toBe(0); // Sunday
    });
});
```

Please note that you'll need to adjust the path to your `findNthWeekdayOfSpecificYear` function according to your project structure. Also, make sure to implement or import the function `findNthWeekdayOfSpecificYear` in your TypeScript file before running these tests.
