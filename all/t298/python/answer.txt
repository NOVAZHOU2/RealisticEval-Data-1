Here's how you can convert the JavaScript function `setEurValue` to Python:

```python
def set_eur_value(value):
    # Check if the input is a valid number
    try:
        num = float(value)
    except ValueError:
        return ""
    
    # Format the number based on its magnitude
    if num < 1000:
        return str(num)
    elif num >= 1000 and num < 1000000:
        return f"{num / 1000:.2f}k"
    else:
        return f"{num / 1000000:.2f}m"

# Example usage:
print(set_eur_value(500))       # Output: "500"
print(set_eur_value(1500))      # Output: "1.5k"
print(set_eur_value(1500000))   # Output: "1.5m"
print(set_eur_value("abc"))     # Output: ""
```

### Explanation:
1. **Input Validation**: The function first tries to convert the input to a float. If it fails (i.e., the input is not a number), it returns an empty string.
2. **Formatting**:
   - If the number is less than 1000, it simply returns the number as a string.
   - If the number is between 1000 and 999,999, it formats the number to two decimal places followed by 'k'.
   - If the number is 1,000,000 or greater, it formats the number to two decimal places followed by 'm'.
