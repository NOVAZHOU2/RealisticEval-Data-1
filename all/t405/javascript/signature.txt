Sure! Below is the equivalent JavaScript function with JSDoc comments:

```javascript
/**
 * Remove the part before the first upper case letter and the first lower case letter from the string.
 * 
 * For example:
 *      input: 1234AbCde5678
 *      output: AbCde5678
 * 
 * @param {...string} strings - Accepts one or more strings as variable arguments
 * @returns {string[]} An array containing the modified strings
 */
function removePartsOfString(...strings) {
    return strings.map(str => {
        const firstUpperCaseIndex = str.search(/[A-Z]/);
        const firstLowerCaseIndex = str.search(/[a-z]/);
        
        // Find the minimum index of the first uppercase and lowercase letters
        const minIndex = Math.min(firstUpperCaseIndex, firstLowerCaseIndex);
        
        // If either index is -1 (meaning no such character was found), use the other index or 0 if neither were found
        const startIndex = minIndex === -1 ? Math.max(firstUpperCaseIndex, firstLowerCaseIndex) : minIndex;
        
        return str.substring(startIndex);
    });
}
```

This function uses the `map` method to apply the transformation to each string in the input array. The `search` method is used to find the indexes of the first uppercase and lowercase letters. The `substring` method then extracts the part of the string starting from the smallest of these indexes.
