Here is the equivalent JavaScript code for your Python function:

```javascript
function removePartsOfString(...strings) {
    return strings.map(str => {
        let firstUpperCaseIndex = str.search(/[A-Z]/);
        let firstLowerCaseIndex = str.search(/[a-z]/);

        if (firstUpperCaseIndex !== -1 && firstLowerCaseIndex !== -1) {
            return str.substring(firstUpperCaseIndex, firstLowerCaseIndex + 1);
        } else {
            return str;
        }
    });
}
```

This JavaScript function does exactly what your Python function does. It accepts any number of string arguments, removes the parts before the first uppercase and lowercase letters in each string, and returns the modified strings. If there's no uppercase or lowercase letter in a string, it simply returns that string unchanged.
