Here is the JavaScript equivalent of your Python function:

```javascript
function cleanDictionary(inputDict) {
  // Initialize an empty object for storing the result
  let result = {};

  // Loop through each key in the input dictionary
  for(let key in inputDict) {
    if(inputDict.hasOwnProperty(key)) {
      // Check if the value is not NaN, not null, not undefined, and not just whitespace
      if(!isNaN(inputDict[key]) && inputDict[key] !== null && inputDict[key] !== undefined && inputDict[key].trim() !== '') {
        // If the value passes all checks, add it to the result dictionary
        result[key] = inputDict[key];
      }
    }
  }

  return result;
}
```

This function works similarly to your Python function. It iterates over each property in the `inputDict` object, checks whether its value is a valid number, not null or undefined, and not just whitespace. If these conditions are met, it adds the property to the `result` object. At the end of the function, it returns the `result` object which contains only the valid properties.
