Here's how you can convert your JavaScript function into TypeScript:

```typescript
/**
 * Counts the occurrences of consecutive identical letters in a given string.
 *
 * @param {string} inputString - The string to analyze for letter changes.
 * @returns {number[]} An array of counts representing the number of consecutive
 *                          identical letters before a different letter is found.
 */
function countLetterChanges(inputString: string): number[] {
  let result: number[] = [];
  let count: number = 1;

  for (let i: number = 0; i < inputString.length; i++) {
    if (inputString[i] === inputString[i + 1]) {
      count++;
    } else {
      result.push(count);
      count = 1;
    }
  }

  return result;
}
```

In TypeScript, we need to define the type of each parameter and return value explicitly. In this case, `inputString` is a `string`, and the function returns an array of numbers (`number[]`). Inside the function, we also declare the types of variables like `result`, `count`, and the loop index `i`.
