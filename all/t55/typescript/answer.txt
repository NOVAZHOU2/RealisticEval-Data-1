Here is the equivalent TypeScript function for your Python code:

```typescript
function minRemovalsToMakeUnique(nums: number[]): number {
    let count = 0;
    const set = new Set<number>();

    for(let i=0; i<nums.length; i++) {
        while(set.has(nums[i])) {
            nums[i]++;
            count++;
        }
        set.add(nums[i]);
    }

    return count;
}
```

This TypeScript function works similarly to your Python function. It uses a `Set` to keep track of unique numbers and increments each number until it becomes unique, counting the total number of increments needed.
