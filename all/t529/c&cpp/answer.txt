Sure, here is your JavaScript function translated into C++:

```cpp
#include <fstream>
#include <sstream>
#include <nlohmann/json.hpp>

using json = nlohmann::json;

void saveAsJSON(const json& data, const std::string& outputFilePath)
{
    std::ofstream outputFile(outputFilePath);
    if (outputFile.is_open())
    {
        outputFile << data.dump(4); // 4 is for indentation
        outputFile.close();
    }
    else 
    {
        std::cerr << "Unable to open file: " << outputFilePath << std::endl;
    }
}
```

In this code, I used `nlohmann/json` library which is a header-only library for parsing JSON in C++. If you don't have this library installed, you can install it using `vcpkg`, `conan`, or download it directly from [here](https://github.com/nlohmann/json).
