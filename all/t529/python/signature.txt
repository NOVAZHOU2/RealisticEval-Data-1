```python
import json

def save_as_json(data, output_file_path):
    """
    Converts the data object to JSON format and saves it to the specified file path.

    :param dict data: The data dictionary to be converted to JSON.
    :param str output_file_path: The file path where the JSON will be saved.
    """
    # Convert the data dictionary to a JSON string
    json_data = json.dumps(data, indent=4)
    
    # Write the JSON string to the specified file path
    with open(output_file_path, 'w') as file:
        file.write(json_data)
```

This Python function `save_as_json` takes a dictionary `data` and a string `output_file_path`, converts the dictionary to a JSON formatted string using `json.dumps`, and then writes that string to the specified file path. The `indent=4` parameter in `json.dumps` makes the JSON output more readable.
