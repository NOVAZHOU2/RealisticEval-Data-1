In the JavaScript version, we use JSDoc comments instead of Python docstrings. The `@param` tag is used to describe the function parameters and their types, while the `@returns` tag describes what the function returns. Note that in JavaScript, arrays are passed by reference, so there's no need for explicit type declarations like in Python.
