Sure! Here is the JavaScript version of the Python function `format_comment` with the same comments:

```javascript
/**
 * Formats a string into a commented block with specified maximum line length.
 *
 * @param {string} string - The input string to format.
 * @param {number} [max_length=60] - Maximum length of each line in the output.
 * @returns {string} A formatted string with each line prefixed by '# ' and not exceeding the specified max_length.
 */
function formatComment(string, maxLength = 60) {
    // Split the input string into words
    const words = string.split(' ');
    let result = '';
    let currentLine = '';

    // Iterate through each word and add it to the current line if it fits within the max_length
    for (let i = 0; i < words.length; i++) {
        if ((currentLine + words[i]).length <= maxLength) {
            currentLine += words[i] + ' ';
        } else {
            result += '# ' + currentLine.trim() + '\n';
            currentLine = words[i] + ' ';
        }
    }

    // Add any remaining text on the last line
    if (currentLine) {
        result += '# ' + currentLine.trim();
    }

    return result;
}
```

This JavaScript function does the same thing as the Python function: it formats a string into a commented block where each line is prefixed by `# ` and does not exceed the specified maximum line length. If the line exceeds the maximum length, it starts a new line with `# `.
