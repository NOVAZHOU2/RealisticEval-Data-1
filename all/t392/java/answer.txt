```java
package org.real.temp;

public class Answer {
    public String lookAndSay(String number) {
        StringBuilder result = new StringBuilder();
        char[] chars = number.toCharArray();
        int count = 1;
        
        for(int i = 1; i < chars.length; i++) {
            if(chars[i] == chars[i - 1]) {
                count++;
            } else {
                result.append(count).append(chars[i - 1]);
                count = 1;
            }
        }

        result.append(count).append(chars[chars.length - 1]);

        return result.toString();
    }
}
```

In this Java version, I've used a `StringBuilder` to build up the result string, and a `for` loop to iterate through each character in the input string. If the current character is the same as the previous one, we increment our count. Otherwise, we append the count and the digit to the result, then reset the count. Finally, we append the last count and digit to the result.
