```typescript
function containsPhoneNumber(s: string): boolean {
    /**
     * Determines whether the string contains a phone number.
     * A possible format for a phone number is +1-800-555-1234, 555-555-1234, or 555 555 1234.
     *
     * @param {string} s - The input string which may contain a phone number.
     * @returns {boolean} - Whether the string contains a phone number.
     */

    // Regular expression pattern for validating phone numbers in the given formats
    const phoneNumberPattern = /^\+?(\d{1,3})?[-.\s]?(\d{3})[-.\s]?(\d{3})[-.\s]?(\d{4})$/;

    return phoneNumberPattern.test(s);
}
```

In this TypeScript version of the function:

- We use a regular expression (`phoneNumberPattern`) to match phone numbers in the specified formats. This pattern allows for optional country codes and various separators between digits.
  
- The `test` method of the RegExp object checks if the input string matches the pattern. If it does, `containsPhoneNumber` returns `true`; otherwise, it returns `false`.
