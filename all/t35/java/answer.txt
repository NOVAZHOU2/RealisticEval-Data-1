Here's the equivalent Java code:

```java
package org.real.temp;

import java.util.List;

public class Answer {

    public static boolean isPointInPolygon(double[] point, List<double[]> polygon) {
        int i, j;
        boolean result = false;
        
        for (i = 0, j = polygon.size() - 1; i < polygon.size(); j = i++) {
            double xi = polygon.get(i)[0];
            double yi = polygon.get(i)[1];
            
            double xj = polygon.get(j)[0];
            double yj = polygon.get(j)[1];
            
            if (((yi > point[1]) != (yj > point[1])) && 
                (point[0] < (xj - xi) * (point[1] - yi) / (yj - yi) + xi)) {
                    result = !result;
            }
        }

        return result;
    }
}
```

Please note that in Java, arrays are used instead of Python's tuple and list types. Also, Java does not have built-in support for optional arguments or type hinting like Python does, so we need to pass all required parameters explicitly.
