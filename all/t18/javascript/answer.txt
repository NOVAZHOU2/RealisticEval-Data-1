```javascript
function floatToRGB(value) {
    /**
     * Convert a floating point number between 0 and 1 to a color from red to green in the color format RGB
     * @param {number} value - A float between 0 and 1.
     * 
     * @returns {Array<number>} An array representing the RGB color.
     */
    if (value < 0 || value > 1) {
        throw new Error('Value must be between 0 and 1');
    }

    let r = Math.floor(255 * (1 - value));
    let g = Math.floor(255 * value);
    let b = 0;

    return [r, g, b];
}
```

This JavaScript function does exactly the same thing as your Python function: it takes a floating-point number between 0 and 1 and converts it to an RGB color. The returned RGB values range from 0 to 255. If the input value is outside of the valid range, an error will be thrown.
