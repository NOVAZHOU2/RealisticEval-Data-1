```python
from datetime import datetime

def calculate_time_difference(given_date):
    if isinstance(given_date, str):
        given_date = datetime.strptime(given_date, '%Y-%m-%d %H:%M:%S')
        
    current_date = datetime.now()
    
    time_difference = current_date - given_date
    
    total_seconds = time_difference.total_seconds()
    minutes = int(total_seconds / 60)
    hours = int(minutes / 60)
    days = int(hours / 24)
    
    remaining_minutes = minutes % 60
    remaining_hours = hours % 24
    
    return {'days': days, 'hours': remaining_hours, 'minutes': remaining_minutes}
```

This function will work for dates in the format `YYYY-MM-DD HH:MM:SS`. If you have dates in another format, you'll need to adjust the format string passed to `strptime()`. 

Also note that this function does not handle cases where `given_date` is in the future. If you want to handle such cases, you can add some additional logic to ensure the returned values make sense.
