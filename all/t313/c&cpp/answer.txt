```cpp
#include <iostream>
#include <QColor>
#include <QWidget>

// Function to detect if the background is too dark or bright
std::string isBackgroundTooDarkOrBright(QWidget* widget) {
    QColor backgroundColor = widget->palette().color(QPalette::Window);

    // Convert RGB values to grayscale
    int grayValue = (backgroundColor.red() * 0.299 + backgroundColor.green() * 0.587 + backgroundColor.blue() * 0.114);

    // Define thresholds for dark and bright
    const int darkThreshold = 60; // Adjust as needed
    const int brightThreshold = 200; // Adjust as needed

    if (grayValue <= darkThreshold) {
        return "dark";
    } else if (grayValue >= brightThreshold) {
        return "bright";
    } else {
        return "normal";
    }
}

int main() {
    // Example usage with a QWidget
    QWidget widget;
    std::cout << isBackgroundTooDarkOrBright(&widget) << std::endl;

    return 0;
}
```

### Explanation:
1. **QColor**: This class represents a color in the RGBA color space.
2. **QWidget**: This class provides the basic infrastructure for creating graphical user interfaces.
3. **isBackgroundTooDarkOrBright**: This function takes a `QWidget` pointer and returns a string indicating whether the background is "dark", "bright", or "normal".
4. **Grayscale Conversion**: The RGB values are converted to a grayscale value using the formula `(R * 0.299 + G * 0.587 + B * 0.114)`.
5. **Thresholds**: The function uses predefined thresholds (`darkThreshold` and `brightThreshold`) to determine the brightness level of the background.
