Here is the JavaScript equivalent of your Python function:

```javascript
function extractOutermostBrackets(s) {
    // Initialize variables
    let stack = [];
    let start = -1;
    let end = -1;

    for(let i=0; i<s.length; i++) {
        if(s[i] === '[') {
            if(stack.length === 0) {
                start = i;
            }
            stack.push('[');
        } else if(s[i] === ']') {
            stack.pop();
            if(stack.length === 0) {
                end = i;
                break;
            }
        }
    }

    // If there were any brackets in the string
    if(start !== -1 && end !== -1) {
        return s.substring(start + 1, end);
    }

    // Return empty string if no brackets found
    return '';
}
```

This JavaScript function works similarly to the Python one. It iterates over each character in the input string, keeping track of the indices of the opening and closing brackets using a stack. When it finds the matching closing bracket for the first opening bracket it encounters, it extracts the substring between them. If no brackets are found, it returns an empty string.
