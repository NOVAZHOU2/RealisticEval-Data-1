```python
import re

def is_valid_password(password):
    # Check for at least one number
    if not re.search(r'\d', password):
        return False
    
    # Check for at least one lowercase letter
    if not re.search(r'[a-z]', password):
        return False
    
    # Check for at least one uppercase letter
    if not re.search(r'[A-Z]', password):
        return False
    
    # Check for at least one punctuation mark
    if not re.search(r'[^\w\s]', password):
        return False
    
    # Check for minimum length of 8 characters
    if len(password) < 8:
        return False
    
    return True
```

This Python function uses regular expressions (the `re` module) to check for each requirement in turn. If any requirement is not met, it returns `False`. If all requirements are met, it returns `True`.
