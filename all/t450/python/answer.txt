```python
import re

def is_valid_password(password):
    """
    Checks whether the provided password meets the specified format requirements:
    - At least one number
    - At least one lowercase letter
    - At least one uppercase letter
    - At least one punctuation mark
    - Minimum length of 8 characters
    
    :param password: str - The password string to validate
    :return: bool - Returns True if the password meets all requirements; otherwise, False.
    """
    
    # Check minimum length
    if len(password) < 8:
        return False
    
    # Check for at least one digit
    if not re.search(r'\d', password):
        return False
    
    # Check for at least one lowercase letter
    if not re.search(r'[a-z]', password):
        return False
    
    # Check for at least one uppercase letter
    if not re.search(r'[A-Z]', password):
        return False
    
    # Check for at least one punctuation mark
    if not re.search(r'[^\w\s]', password):
        return False
    
    return True

# Example usage:
password = "Example123!"
print(is_valid_password(password))  # Output: True or False based on the password
```

This Python function uses regular expressions (`re` module) to check each requirement for the password. It ensures that the password contains at least one digit, one lowercase letter, one uppercase letter, one punctuation mark, and has a minimum length of 8 characters.
