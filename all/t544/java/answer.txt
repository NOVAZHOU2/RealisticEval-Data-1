Here is the equivalent Java code for your JavaScript function:

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

/**
 * Filters out all even numbers from an array.
 *
 * @param array The array of numbers to filter.
 * @return A new list containing only the odd numbers.
 */
public List<Integer> filterOutEvenNumbers(Integer[] array) {
    List<Integer> result = new ArrayList<>();
    for (Integer num : array) {
        if (num % 2 != 0) {
            result.add(num);
        }
    }
    return result;
}
```

This Java method does exactly what your JavaScript function does: it filters out all even numbers from an integer array and returns a new list with only the odd numbers.

Please note that in Java, arrays are zero-indexed just like in JavaScript, and you need to use `List` instead of arrays when you want dynamic resizing or more methods related to collections. In this case, we're using `ArrayList`.
