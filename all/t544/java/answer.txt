```java
import java.util.ArrayList;
import java.util.List;

/**
 * Filters out all even numbers from an array.
 *
 * @param array - The array of numbers to filter.
 * @return A new list containing only the odd numbers.
 */
public List<Integer> filterOutEvenNumbers(Integer[] array) {
    List<Integer> result = new ArrayList<>();
    for (Integer num : array) {
        if (num % 2 != 0) {
            result.add(num);
        }
    }
    return result;
}
```
This function takes an array of Integer objects and returns a new list that contains only the odd numbers from the original array. It uses a for-each loop to iterate over each number in the input array, checks if it's not divisible by 2 (i.e., it's odd), and if so, adds it to the result list.
