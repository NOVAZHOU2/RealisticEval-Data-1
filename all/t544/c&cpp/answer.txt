```cpp
#include <vector>
#include <algorithm>

/**
 * Filters out all even numbers from a vector.
 *
 * @param array - The vector of integers to filter.
 * @return A new vector containing only the odd numbers.
 */
std::vector<int> filterOutEvenNumbers(const std::vector<int>& array) {
    std::vector<int> result;
    std::copy_if(array.begin(), array.end(), std::back_inserter(result), [](int num) {
        return num % 2 != 0;
    });
    return result;
}
```

1. **Include Necessary Headers**: We include `<vector>` for using vectors and `<algorithm>` for algorithms like `std::copy_if`.

3. **Result Vector**: We create an empty vector called `result` to store the odd numbers.
4. **Copy If Algorithm**: We use `std::copy_if` to copy elements from the input vector to the result vector if they satisfy the condition (i.e., they are odd). The lambda function `[](int num) { return num % 2 != 0; }` checks if a number is odd.
5. **Return Result**: Finally, we return the `result` vector.
