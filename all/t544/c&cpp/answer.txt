```cpp
#include <vector>
#include <algorithm>

/**
 * Filters out all even numbers from a vector.
 *
 * @param array - The vector of numbers to filter.
 * @return A new vector containing only the odd numbers.
 */
std::vector<int> filterOutEvenNumbers(const std::vector<int>& array) {
    std::vector<int> result;
    std::copy_if(array.begin(), array.end(), std::back_inserter(result),
                 [](int num) { return num % 2 != 0; });
    return result;
}
```

### Explanation:
1. **Include Necessary Headers**: We include `<vector>` for using vectors and `<algorithm>` for algorithms like `copy_if`.
2. **Function Signature**: The function `filterOutEvenNumbers` takes a constant reference to a `std::vector<int>` and returns a `std::vector<int>`.
3. **Copy If Algorithm**: We use `std::copy_if` to copy elements from the input vector to the result vector if they satisfy the condition (i.e., `num % 2 != 0`, which checks if the number is odd).
4. **Lambda Function**: The lambda function `[&](int num)` is used to check if each number in the vector is odd.

This C++ code will achieve the same functionality as your JavaScript function, filtering out all even numbers from an array and returning a new array with only the odd numbers.
