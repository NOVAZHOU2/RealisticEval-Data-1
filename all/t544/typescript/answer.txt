```typescript
/**
 * Filters out all even numbers from an array.
 *
 * @param {number[]} array - The array of numbers to filter.
 * @returns {number[]} - A new array containing only the odd numbers.
 */
function filterOutEvenNumbers(array: number[]): number[] {
    return array.filter(num => num % 2 !== 0);
}
```

In this TypeScript version, I've added type annotations for both the input parameter and the return value of the `filterOutEvenNumbers` function. Also, I've used the built-in `.filter()` method on the array which returns a new array with all elements that pass the test implemented by the provided function. This function checks if a number is not divisible by 2 (`num % 2 !== 0`).
