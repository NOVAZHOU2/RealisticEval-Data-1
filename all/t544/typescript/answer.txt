Sure! Here's how you can convert your JavaScript function into TypeScript:

```typescript
/**
 * Filters out all even numbers from an array.
 *
 * @param {number[]} array - The array of numbers to filter.
 * @returns {number[]} - A new array containing only the odd numbers.
 */
function filterOutEvenNumbers(array: number[]): number[] {
  return array.filter(num => num % 2 !== 0);
}
```

In this TypeScript version:
- The `array` parameter is explicitly typed as `number[]`.
- The function returns a `number[]`.

The `filter` method is used to create a new array with all elements that pass the test implemented by the provided function (`num % 2 !== 0` checks if the number is odd).
