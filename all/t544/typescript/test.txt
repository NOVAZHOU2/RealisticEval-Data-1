1. **Initialize a new Node.js project**:
   ```bash
   mkdir my-project
   cd my-project
   npm init -y
   ```

2. **Install TypeScript and Jest**:
   ```bash
   npm install --save-dev typescript ts-jest @types/jest
   ```

3. **Create a `tsconfig.json` file**:
   ```json
   {
     "compilerOptions": {
       "target": "ES6",
       "module": "commonjs",
       "strict": true,
       "esModuleInterop": true,
       "skipLibCheck": true,
       "forceConsistentCasingInFileNames": true
     },
     "include": ["src"]
   }
   ```

4. **Create a Jest configuration file** (`jest.config.js`):
   ```javascript
   module.exports = {
     preset: 'ts-jest',
     testEnvironment: 'node',
   };
   ```

5. **Create a directory structure**:
   ```
   my-project/
   ├── src/
   │   └── filterOutEvenNumbers.ts
   ├── __tests__/
   │   └── filterOutEvenNumbers.test.ts
   ├── package.json
   ├── tsconfig.json
   └── jest.config.js
   ```

6. **Run your tests**:
   ```bash
   npx jest
   ```
