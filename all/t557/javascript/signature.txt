```javascript
/**
 * Convert an angle from radians to degrees.
 *
 * @param {number} radians - The angle in radians to convert.
 * @returns {number} The angle in degrees.
 */
function radiansToDegrees(radians) {
    return radians * (180 / Math.PI);
}
```

This JavaScript function `radiansToDegrees` takes a single argument `radians` and returns the equivalent angle in degrees. The conversion formula used is \( \text{degrees} = \text{radians} \times \frac{180}{\pi} \).
