Sure! Below is the equivalent JavaScript code using the `mammoth` library to extract text from a `.docx` file:

First, you need to install the `mammoth` library. You can do this using npm:

```bash
npm install mammoth
```

Then, you can use the following JavaScript function to extract text from a `.docx` file:

```javascript
const mammoth = require("mammoth");

async function extractTextFromWord(docxFilePath) {
    /**
     * Extracts text content from a given Word file (.docx).
     *
     * @param {string} docxFilePath - The path to the Word file.
     * @returns {Promise<string>} The extracted text content.
     */
    
    try {
        const result = await mammoth.convertToHtml({ path: docxFilePath });
        return result.value; // The generated HTML
    } catch (error) {
        console.error("Error extracting text from .docx file:", error);
        throw error;
    }
}

// Example usage:
extractTextFromWord('path/to/your/file.docx')
    .then(text => {
        console.log(text);
    })
    .catch(error => {
        console.error(error);
    });
```

This JavaScript function uses the `mammoth` library to convert the `.docx` file to HTML and then extracts the text content from the HTML. Make sure to replace `'path/to/your/file.docx'` with the actual path to your `.docx` file.
