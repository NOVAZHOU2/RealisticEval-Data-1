Here is how you can write your Python code in C++:

```c++
#include <iostream>
#include <string>
#include <sstream>

std::istringstream stream;
std::string word;

void wrap_content_generator(const std::string &content, int width = 80) {
    stream.clear();
    stream.str(content);
    
    while(getline(stream, word, ' ')) {
        if(word.length() > width) {
            for(int i = 0; i < word.length(); i++) {
                if(i % width == 0 && i != 0)
                    std::cout << '\n';
                std::cout << word[i];
            }
        } else {
            std::cout << word << ' ';
        }
    }
}
```

This C++ function does exactly what your Python function does. It takes a string `content` and an optional integer `width`, then prints out each word from `content` on a new line when it reaches the specified `width`.

Please note that the C++ version doesn't handle multi-line strings or complex text formatting like Python's `textwrap`. If you need those features in C++, you might want to use a library like Boost.TextProcessing or implement them yourself.
