```python
import urllib.parse

def get_file_id_from_url(url):
    """
    Extracts the fileid from the given URL query args. If not found returns None.
    
    For example:
        input: https://example.com/download?fileId=12345
        output: '12345'

    :param str url: The URL from which the file ID is to be extracted.
    :return: The extracted file ID if present, otherwise None if the URL does not conform to the expected format.
    :rtype: str or None
    """
    parsed_url = urllib.parse.urlparse(url)
    query_params = urllib.parse.parse_qs(parsed_url.query)

    # Check if 'fileId' key exists in the parsed query parameters and return its value
    return query_params.get('fileId', [None])[0]
```

This Python function uses `urllib.parse` module to parse the URL and extract the query parameters. It then checks for the presence of 'fileId' in these parameters and returns its value if it exists; otherwise, it returns `None`.
