Sure! Here's how you can convert your JavaScript function to TypeScript:

```typescript
/**
 * Checks whether a file name is a C++ header file.
 *
 * @param {string} fileName - The name of the file to check.
 * @returns {boolean} - Returns true if the file is a C++ header file, false otherwise.
 */
function isCppHeaderFile(fileName: string): boolean {
    return fileName.endsWith('.h') || fileName.endsWith('.hpp');
}
```

In this TypeScript version:
- The `fileName` parameter is explicitly typed as `string`.
- The function's return type is explicitly typed as `boolean`.
- The function checks if the `fileName` ends with either `.h` or `.hpp`, which are common extensions for C++ header files.
