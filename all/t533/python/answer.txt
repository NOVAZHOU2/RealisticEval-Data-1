```python
import random

def shuffle_string(input_string):
    """
    Shuffles the characters in a given string randomly.

    :param input_string: The string to shuffle.
    :return: A new string with the characters shuffled.
    """
    return ''.join(random.sample(input_string, len(input_string)))
```

This function uses the `random` module's `sample()` function to get a list of the input string's characters in a random order and then joins them back together into a string.
