Sure! Here's how you can convert the JavaScript function `shuffleString` to TypeScript:

```typescript
/**
 * Shuffles the characters in a given string randomly.
 *
 * @param {string} inputString - The string to shuffle.
 * @returns {string} A new string with the characters shuffled.
 */
function shuffleString(inputString: string): string {
    let array = inputString.split('');
    for (let i = array.length - 1; i > 0; i--) {
        const j = Math.floor(Math.random() * (i + 1));
        [array[i], array[j]] = [array[j], array[i]];
    }
    return array.join('');
}
```

In this TypeScript version:
- I've added type annotations for the `inputString` parameter and the return value.
- The body of the function remains the same, but now it is compatible with TypeScript's type system.
