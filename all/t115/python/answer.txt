def sort_by_key(array, key):
    """
    Sorts a list of dictionaries alphabetically by a specified key.

    :param array: List of dictionaries to be sorted.
    :param key: Key in the dictionaries to sort by.
    :return: The sorted list based on the specified key.
    """
    return sorted(array, key=lambda x: (str(x.get(key, '')).lower()))

# Example usage:
data = [
    {"name": "Alice", "age": 30},
    {"name": "bob", "age": 25},
    {"name": "Charlie", "age": 35},
]

sorted_data = sort_by_key(data, "name")
print(sorted_data)
Explanation:
Function Definition: The function sort_by_key takes two parameters: array, a list of dictionaries, and key, the key to sort by.
Sorting Logic: The sorted() function is used with a custom key function defined using a lambda. This lambda retrieves the value for the specified key, converts it to a string, and then to lowercase for case-insensitive sorting.
Return Value: The function returns the sorted list of dictionaries.
Example Usage:
In the example usage, a list of dictionaries is sorted by the "name" key, demonstrating how to call the function. The sorted result is printed to the console.
