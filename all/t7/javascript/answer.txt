Here's how you can convert your Python code into JavaScript:

```javascript
const { createLogger, format, transports } = require('winston');

class Logger {
  constructor(name, level = 'debug') {
    /**
     * Initializes a new logger instance.
     *
     * @param {string} name - Name of the logger, typically __name__ to reference the module name.
     * @param {string} level - Logging level, default is debug.
     */

    this.logger = createLogger({
      level,
      format: format.json(),
      transports: [
        new transports.Console()
      ]
    });
  }

  log(level, message) {
    /**
     * Logs a message with the given level.
     *
     * @param {string} level - Logging level for the message (e.g., info).
     * @param {string} message - Log message.
     */
    
    this.logger.log({ level, message });
  }
}
```

This JavaScript code uses the `winston` library for logging, which is similar to Python's `logging` module. The `Logger` class in JavaScript has a constructor that takes a name and a level parameter, just like in the Python version. It also has a `log` method that logs messages at the specified level. 

Remember to install the `winston` package before running this code using npm (`npm install winston`).
