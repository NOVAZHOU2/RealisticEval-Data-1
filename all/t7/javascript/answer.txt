Sure! Here's the equivalent JavaScript code for your Python logger:

```javascript
class Logger {
    constructor(name, level = 'DEBUG') {
        /**
         * Initializes a new logger instance.
         *
         * @param name - Name of the logger.
         * @param level - Logging level, default is DEBUG.
         */
        this.name = name;
        this.level = level;
    }

    log(level, message) {
        /**
         * Logs a message with the given level.
         *
         * @param level - Logging level for the message.
         * @param message - Log message.
         */
        console.log(`[${this.name}] [${level}] ${message}`);
    }
}
```
