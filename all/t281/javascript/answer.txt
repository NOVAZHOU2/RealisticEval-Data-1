```javascript
function squaredEuclideanDistance(vec1, vec2) {
    if (vec1.length !== vec2.length) {
        throw new Error('Vectors must have same length');
    }

    let sum = 0;
    for(let i=0; i<vec1.length; i++) {
        sum += Math.pow(vec1[i] - vec2[i], 2);
    }
    
    return sum;
}
```

This function will calculate the squared Euclidean distance between two vectors in JavaScript. The `Math.pow` method is used to raise a number to the power of another number. If the lengths of the input vectors are not equal, an error is thrown.
