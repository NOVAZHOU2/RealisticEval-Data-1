Here's how you can write the Python function `camel_to_snake` in Java:

```java
package org.real.temp;

public class Answer {
    
    /**
     * Convert a CamelCase string to snake_case.
     *
     * @param camelStr The CamelCase string to convert.
     * @return The converted snake_case string.
     */
    public static String camelToSnake(String camelStr) {
        if (camelStr == null || camelStr.isEmpty()) {
            return "";
        }
        
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < camelStr.length(); i++) {
            char c = camelStr.charAt(i);
            if (Character.isUpperCase(c)) {
                sb.append("_").append(Character.toLowerCase(c));
            } else {
                sb.append(c);
            }
        }
        return sb.toString();
    }
}
```

This Java method does the same thing as your Python function - it converts a CamelCase string to snake_case. It iterates over each character of the input string and appends an underscore followed by the lowercase version of the character if the current character is uppercase. If the character is already lowercase or not alphabetic, it just appends it directly to the result.
