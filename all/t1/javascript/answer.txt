Here's how you can convert your Python function into JavaScript:

```javascript
function numericalStrConvert(value) {
  // check if value is an integer
  const intValue = parseInt(value);
  if (!isNaN(intValue)) {
    return intValue;
  }

  // check if value is a floating point number
  const floatValue = parseFloat(value);
  if (!isNaN(floatValue)) {
    return floatValue;
  }

  // if neither, return the original string
  return value;
}
```

This JavaScript function works similarly to your Python function. It attempts to parse the input string as an integer and returns that integer if successful. If not, it tries to parse the string as a floating-point number and returns that number if successful. If both parsing attempts fail, it simply returns the original string.
