package org.real.temp;

public class Answer {

    public static Object numericalStrConvert(String value) {
        try {
            return Integer.parseInt(value);
        } catch (NumberFormatException e1) {
            try {
                return Double.parseDouble(value);
            } catch (NumberFormatException e2) {
                return value;
            }
        }
    }

}

This Java method tries to parse the input string into an `Integer`, and if that fails, it tries to parse it into a `Double`. If both parsing attempts fail, it simply returns the original string.

Note that in Java, there isn't a direct equivalent of Python's `Union` type hinting. Instead, we use the `Object` type which can represent any kind of data. The actual type returned will be either `Integer`, `Double`, or `String`.
