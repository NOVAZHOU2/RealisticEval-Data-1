Here is the equivalent JavaScript code for your Python function:

```javascript
function rgbToHsv(r, g, b) {
    // Convert RGB to [0, 1] range
    r /= 255;
    g /= 255;
    b /= 255;

    let max = Math.max(r, g, b), min = Math.min(r, g, b);
    let h, s, v = max;

    let d = max - min;
    s = max == 0 ? 0 : d / max;

    if(max == min){
        h = 0; // achromatic
    }else{
        switch(max){
            case r: h = (g - b) / d + (g < b ? 6 : 0); break;
            case g: h = (b - r) / d + 2; break;
            case b: h = (r - g) / d + 4; break;
        }
        h /= 6;
    }

    return [Math.round(h * 360), Math.round(s * 100), Math.round(v * 100)];
}
```

This function takes three parameters `r`, `g` and `b` which represent red, green and blue values respectively. It returns an array of three integers representing hue, saturation and value in HSV format.

Please note that unlike Python, JavaScript doesn't have built-in support for tuples or type hinting, so we're returning an array instead. The values are also returned as integers, as JavaScript does not differentiate between integer and float types.
