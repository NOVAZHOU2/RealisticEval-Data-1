Here is the equivalent C++ code signature with the same comments:

```cpp
#include <vector>
#include <cmath>

/**
 * Generates the ith row of Pascal's Triangle.
 *
 * @param i The row index (0-indexed)
 * @return A vector representing the ith row of Pascal's Triangle
 */
std::vector<int> pascalTriangleRow(int i);
```

Note that in C++, we use `std::vector` instead of Python's built-in `List`. Also, since C++ doesn't have type hinting like Python, we specify the return type and parameter types directly in the function declaration.
