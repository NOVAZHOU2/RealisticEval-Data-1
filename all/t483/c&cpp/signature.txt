Here's how you can write the same Python function in C++:

```cpp
#include <regex>
#include <string>

/**
 * @brief Verifies if the provided string is a valid email address.
 *
 * @param email The email address to validate.
 * @return true If the email address is valid.
 * @return false If the email address is not valid.
 */
bool isValidEmail(const std::string& email)
{
    // Define the regular expression for validating an Email
    std::regex pattern(R"((\w+)([\-+.']\w+)*@\w+([\-+.']\w+)*\.\w+([\-+.']\w+)*)");

    // Check if the email matches the pattern
    return std::regex_match(email, pattern);
}
```

This C++ code uses the `<regex>` library to define and match against a regular expression that represents the general format of an email address. It returns `true` if the email matches the pattern, which indicates it is valid, or `false` otherwise. Note that error handling isn't included here; in a production environment, you'd want to add appropriate error checking.
