```c++
#include <regex>
#include <string>

bool is_valid_email(const std::string &email)
{
    // Regular expression for validating an Email
    std::regex pattern(R"((\w+)(\.\w+)*@\w+(\.\w+)*)");

    // Check if the email matches the pattern
    return std::regex_match(email, pattern);
}
```

This function uses the `<regex>` library in C++, which provides support for regular expressions. The `is_valid_email` function checks whether the input email string matches the specified regular expression pattern. If it does, the function returns true; otherwise, it returns false.

Note that this is a very basic validation and may not cover all possible valid email addresses according to RFC 5322. For more comprehensive validation, consider using a more complex regular expression or a dedicated library.
