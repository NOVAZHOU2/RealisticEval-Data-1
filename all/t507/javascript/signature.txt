Here's the JavaScript version of your Python function:

```javascript
/**
 * Check if the provided password is strong.
 *
 * A strong password must satisfy the following criteria:
 * - At least one lowercase letter
 * - At least one uppercase letter
 * - At least one number
 * - At least 8 characters long
 *
 * @param {string} password - The password to check.
 * @returns {boolean} - True if the password is strong, False otherwise.
 */
function isStrongPassword(password) {
    // Regular expression for checking at least one lowercase letter
    const hasLower = /[a-z]/.test(password);
    
    // Regular expression for checking at least one uppercase letter
    const hasUpper = /[A-Z]/.test(password);
    
    // Regular expression for checking at least one digit
    const hasDigit = /\d/.test(password);
    
    // Checking if the password length is at least 8
    const lengthCheck = password.length >= 8;

    return hasLower && hasUpper && hasDigit && lengthCheck;
}
```

In this JavaScript function, I used regular expressions to check for the presence of lowercase letters, uppercase letters, and digits in the input string. Also, I checked if the length of the password is at least 8. The `&&` operator ensures that all conditions have to be true for the function to return `true`. If any condition fails, it will return `false`.
