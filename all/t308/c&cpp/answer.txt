```cpp
#include <iostream>
#include <iomanip>
#include <ctime>

/**
 * Gets the current date and returns it as YYYY-MM-DD
 *
 * @return std::string The current date formatted as YYYY-MM-DD.
 */
std::string getCurrentDate() {
    // Get the current time
    std::time_t now = std::time(nullptr);
    std::tm* localTime = std::localtime(&now);

    // Create an output string stream
    std::ostringstream oss;
    
    // Format the date as YYYY-MM-DD
    oss << std::put_time(localTime, "%Y-%m-%d");

    return oss.str();
}

int main() {
    // Example usage of the getCurrentDate function
    std::cout << "Current Date: " << getCurrentDate() << std::endl;

    return 0;
}
```

### Explanation:
1. **Include Necessary Headers**: We include `<iostream>`, `<iomanip>`, and `<ctime>` to handle input/output operations, formatting, and time-related functions respectively.

2. **Function Definition**:
   - The `getCurrentDate` function uses `std::time(nullptr)` to get the current time in seconds since the epoch.
   - `std::localtime(&now)` converts the time to a local time structure (`std::tm`).
   - An `std::ostringstream` object is used to format the date as `YYYY-MM-DD`.
   - `std::put_time(localTime, "%Y-%m-%d")` formats the time according to the specified format.
   - Finally, the formatted date string is returned.

3. **Main Function**:
   - The `main` function demonstrates how to use the `getCurrentDate` function and prints the current date.

This C++ code will give you the current date in the format `YYYY-MM-DD`.
