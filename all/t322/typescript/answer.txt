Sure! Here's how you can convert the JavaScript function to TypeScript:

```typescript
/**
 * Verify that a string is a valid email address.
 *
 * @param {string} email - The email address to be validated.
 * @returns {boolean} - Returns true if the email matches the regex pattern, indicating it is valid,
 *                      or false otherwise.
 */
function isValidEmail(email: string): boolean {
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return emailRegex.test(email);
}
```

In this TypeScript version:
- I've added type annotations for the `email` parameter and the return value of the function.
- I've used a regular expression to validate the email format. This regex checks for a typical email structure with characters before and after an "@" symbol and a domain suffix.
