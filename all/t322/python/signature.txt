```python
import re

/**
 * verify that a string is a valid email address
 *
 * @param {str} email - The email address to be validated.
 * @return {bool} - Returns True if the email matches the regex pattern, indicating it is valid,
 *                   or False otherwise.
 */
def is_valid_email(email):
    # Regular expression for validating an Email
    regex = r'^[a-z0-9]+[\._]?[a-z0-9]+[@]\w+[.]\w{2,3}$'
    
    # If the string is empty
    if email is None:
        return False
    
    # Matching the regex to the email
    if re.search(regex, email):
        return True
    else:
        return False
```

This Python function `is_valid_email` checks if the provided email string matches a regular expression pattern designed to validate email addresses. It returns `True` if the email is valid according to the pattern, and `False` otherwise.
