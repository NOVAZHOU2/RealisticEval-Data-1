Here's how you can convert your Python code into TypeScript:

```typescript
function containsEmail(text: string): boolean {
    /**
     * Check if the given text contains an email address.
     *
     * @param {string} text - The string to search for an email address.
     * @returns {boolean} - True if an email address is found, False otherwise.
     */

    // Regular expression pattern for matching email addresses
    const emailRegex = /\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b/;

    return emailRegex.test(text);
}
```

In this TypeScript version of the function:

1. We define a function `containsEmail` that takes a single parameter `text` of type `string`.
2. We use a regular expression (`emailRegex`) to match email addresses in the input text.
3. We utilize the `test` method of the RegExp object to check if the input text matches the email regex pattern.
4. The function returns a boolean value indicating whether an email address was found or not.
