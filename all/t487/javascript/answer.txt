```javascript
function containsEmail(text) {
    /**
     * Check if the given text contains an email address.
     *
     * @param {string} text - The string to search for an email address.
     * @returns {boolean} - True if an email address is found, False otherwise.
     */

    // Regular expression to match an email address
    const emailRegex = /\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b/;

    // Test the input text against the regular expression
    return emailRegex.test(text);
}
```

### Explanation:
1. **Function Definition**: The `containsEmail` function takes one parameter `text`.
2. **Regular Expression**: A regular expression (`emailRegex`) is defined to match an email address.
3. **Testing**: The `test` method of the regular expression object is used to check if the input `text` matches the pattern.
4. **Return Value**: The function returns `true` if an email address is found, and `false` otherwise.

This JavaScript function behaves similarly to your Python function in checking for the presence of an email address in a given string.
