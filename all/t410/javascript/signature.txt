Here's how you can convert your Python code signature to JavaScript:

```javascript
/**
 * Checks the XOR sums of specific columns in a given combination array.
 *
 * @param {Array<Array<number>>} combination - A 2D array where each sub-array corresponds
 *                                            to a specific value.
 * @returns {boolean} True if the XOR sums of the specified columns match the required
 *                    values; otherwise, false.
 */
```

Note that JavaScript does not have built-in support for arrays with typed elements like NumPy does in Python. Therefore, we use `Array<Array<number>>` to represent a 2D array. Also, JavaScript doesn't have type annotations like Python, so we don't specify types for parameters and return values directly in the function signature. Instead, we document them using JSDoc style comments.
