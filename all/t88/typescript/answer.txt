To convert the provided JavaScript function to TypeScript, you'll want to add type annotations to the parameters and potentially the return type. Here's how you can do that:

```typescript
function isCronBetween2And4AM(cronExpression: string): boolean {
    const parts = cronExpression.split(' ');
    const hourPart = parts[1];

    const hours = hourPart.split(',').map(Number);
    return hours.some(hour => hour >= 2 && hour < 4);
}
```

### Key Changes:
1. **Parameter Type Annotation**: The parameter `cronExpression` is annotated as a `string`.
2. **Return Type Annotation**: The function's return type is specified as `boolean`.

This provides better type safety and clarity, which are key benefits of using TypeScript.
