```java
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class CronChecker {

    public static boolean isCronBetween2And4AM(String cronExpression) {
        String[] parts = cronExpression.split(" ");
        String hourPart = parts[1];

        List<Integer> hours = Arrays.stream(hourPart.split(","))
                                    .map(Integer::parseInt)
                                    .collect(Collectors.toList());

        return hours.stream().anyMatch(hour -> hour >= 2 && hour < 4);
    }

    public static void main(String[] args) {
        // Example usage
        String cronExpression = "0 2,3 * * *";
        System.out.println(isCronBetween2And4AM(cronExpression)); // Should print true
    }
}
```

### Explanation:
1. **Splitting the Cron Expression**:
   - The `cronExpression` is split into an array of strings using `" "` (space) as the delimiter.
   - The second element (`parts[1]`) represents the hour part.

2. **Parsing Hours**:
   - The hour part is further split by commas and converted to a list of integers using `Arrays.stream` and `Integer::parseInt`.

3. **Checking Hours**:
   - The list of hours is checked using a stream operation `hours.stream().anyMatch` to see if any hour falls between 2 and 4 AM (inclusive).

4. **Main Method**:
   - A simple example usage is provided in the `main` method to demonstrate how the function works.
