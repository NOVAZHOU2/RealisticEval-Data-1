```javascript
function wordFilterCounter(text, filterWords) {
    let regex = new RegExp(`\\b(${filterWords.join('|')})\\b`, 'gi');
    let matches = text.match(regex);
    let counter = {};
    
    if(matches){
        for(let i=0; i<matches.length; i++){
            if(counter[matches[i]]){
                counter[matches[i]]++;
            } else {
                counter[matches[i]]=1;
            }
        }
    }

    return Object.keys(counter).reduce((acc, key) => {
        acc[key] = counter[key];
        return acc;
    }, {});
}
```

This JavaScript function does the same thing as your Python function. It takes a string (`text`) and an array of strings (`filterWords`). Then it creates a regular expression to match any of the `filterWords` in the `text`. If there are matches, it counts how many times each one appears and returns an object with the counts. If no matches are found, it returns an empty object.
