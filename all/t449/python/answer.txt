Sure! Here is the equivalent Python function that converts a string to Base64 encoding:

```python
import base64

def convert_to_base64(input_string):
    """
    Converts a string to Base64 encoding.

    :param input_string: The string to be converted to Base64.
    :return: The Base64 encoded string.
    """
    # Encode the input string to bytes
    input_bytes = input_string.encode('utf-8')
    
    # Convert the bytes to Base64
    base64_bytes = base64.b64encode(input_bytes)
    
    # Decode the Base64 bytes back to string
    base64_string = base64_bytes.decode('utf-8')
    
    return base64_string
```

This Python function `convert_to_base64` takes an input string, encodes it to bytes using UTF-8 encoding, then uses the `base64.b64encode` function to convert these bytes to Base64. Finally, it decodes the Base64 bytes back to a string and returns it.
