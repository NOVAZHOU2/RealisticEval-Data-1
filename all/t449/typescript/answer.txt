```typescript
/**
 * Converts a string to Base64 encoding.
 *
 * @param {string} input - The string to be converted to Base64.
 * @returns {string} The Base64 encoded string.
 */
function convertToBase64(input: string): string {
    // Create a Buffer from the input string
    const buffer = Buffer.from(input, 'utf-8');
    // Convert the Buffer to a Base64 encoded string
    return buffer.toString('base64');
}
```

In this TypeScript version, I've added type annotations to the `input` parameter and the return type of the function.
