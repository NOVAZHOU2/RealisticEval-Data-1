Here is the equivalent Java code for your JavaScript function:

```java
import java.util.Base64;

public class Main {

    /**
     * Converts a string to Base64 encoding.
     *
     * @param input - The string to be converted to Base64.
     * @return The Base64 encoded string.
     */
    public static String convertToBase64(String input) {
        return Base64.getEncoder().encodeToString(input.getBytes());
    }

    public static void main(String[] args) {
        System.out.println(convertToBase64("Hello World"));
    }
}
```

This Java code uses the built-in `Base64` encoder from the `java.util` package to encode the input string into Base64 format. The `convertToBase64()` method returns the Base64 encoded string.

The `main()` method demonstrates how to use the `convertToBase64()` method with an example string "Hello World". It prints the resulting Base64 encoded string to the console.
