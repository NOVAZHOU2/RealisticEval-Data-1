Sure! Here is the equivalent Java code with the same functionality and comments:

```java
/**
 * Converts a string to Base64 encoding.
 *
 * @param input The string to be converted to Base64.
 * @return The Base64 encoded string.
 */
public String convertToBase64(String input) {
    // Convert the input string to bytes
    byte[] inputBytes = input.getBytes();
    
    // Encode the bytes to Base64
    String base64EncodedString = java.util.Base64.getEncoder().encodeToString(inputBytes);
    
    return base64EncodedString;
}
```

This Java method `convertToBase64` takes a string as input, converts it to bytes, encodes those bytes using Base64 encoding, and returns the resulting Base64 encoded string.
