Here is how you can convert your Python code to JavaScript using the 'array' module from Node.js:

```javascript
function flipPointCloud(pointCloud, axis) {
    // Check if input is an Array and length is at least 3
    if (!Array.isArray(pointCloud) || pointCloud.length < 3) {
        throw new Error('Input must be an array with at least three elements.');
    }

    let flippedPointCloud = [];
    
    for(let i=0; i<pointCloud.length; i++){
        let flippedPoint = [...pointCloud[i]]; // Spread operator to create a copy
        flippedPoint[axis] *= -1; // Multiply the selected axis by -1 to flip it
        flippedPointCloud.push(flippedPoint);
    }
    
    return flippedPointCloud;
}
```

This JavaScript function does exactly the same thing as your Python function. It flips each point in the given point cloud across the specified axis. Note that arrays in JavaScript are zero-indexed just like Python lists. So, to flip along the x-axis, you would call `flipPointCloud(pointCloud, 0)`, for the y-axis `flipPointCloud(pointCloud, 1)` and for the z-axis `flipPointCloud(pointCloud, 2)`.
