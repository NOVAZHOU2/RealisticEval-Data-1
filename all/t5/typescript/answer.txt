```typescript
function matrixMultiply(matrixA: number[][], matrixB: number[][]): number[][] {
    // Check if matrices can be multiplied
    if (matrixA[0].length !== matrixB.length) {
        throw new Error('Number of columns in matrixA must equal number of rows in matrixB');
    }

    let result: number[][] = Array.from({ length: matrixA.length }, () => Array(matrixB[0].length).fill(0));

    for(let i=0; i<matrixA.length; i++) {
        for(let j=0; j<matrixB[0].length; j++) {
            for(let k=0; k<matrixB.length; k++) {
                result[i][j] += matrixA[i][k] * matrixB[k][j];
            }
        }
    }

    return result;
}
```

This TypeScript function `matrixMultiply` takes two 2D arrays (`matrixA` and `matrixB`) as arguments and returns their product. It first checks if the number of columns in `matrixA` equals the number of rows in `matrixB`, which is a requirement for matrix multiplication. If not, it throws an error.

The resulting matrix is initialized with zeros using `Array.from`. Then, three nested loops are used to calculate each element of the result matrix by summing up products of corresponding elements from `matrixA` and `matrixB`.

Please note that unlike Python's list comprehensions, JavaScript/TypeScript does not support them directly for creating 2D arrays filled with initial values, so we use a combination of `Array.from` and `fill()` instead.
