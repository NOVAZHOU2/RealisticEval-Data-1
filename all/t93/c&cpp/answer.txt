Sure! Here is the equivalent C++ code that generates a character array of length 52 containing all lowercase and uppercase letters in alphabetical order:

```cpp
#include <iostream>
#include <vector>
#include <string>

// Function to produce a character array of length 52 containing all lowercase and uppercase letters
std::vector<char> getAllAlphabets() {
    const int alphabetCount = 26; // Number of letters in the English alphabet
    std::vector<char> result(alphabetCount * 2);

    for (int i = 0; i < alphabetCount * 2; ++i) {
        // Determine the character code based on the index:
        // - If the index is less than 26, it's a lowercase letter ('a' to 'z')
        // - If the index is 26 or greater, it's an uppercase letter ('A' to 'Z')
        char charCode = i < alphabetCount
            ? 'a' + i  // Lowercase letters start at char code 'a'
            : 'A' + i - alphabetCount; // Uppercase letters start at char code 'A'

        // Assign the character to the vector
        result[i] = charCode;
    }

    return result;
}

// Function to print the generated alphabets
void printAlphabets(const std::vector<char>& alphabets) {
    for (char c : alphabets) {
        std::cout << c << ' ';
    }
    std::cout << std::endl;
}

int main() {
    std::vector<char> alphabets = getAllAlphabets();
    printAlphabets(alphabets);
    return 0;
}
```

### Explanation:
1. **Function `getAllAlphabets`**:
   - Initializes a vector `result` with a size of 52.
   - Uses a loop to fill the vector with lowercase and uppercase letters.
   - The character codes are determined based on the index.

2. **Function `printAlphabets`**:
   - Prints the characters in the vector.

3. **Main function**:
   - Calls `getAllAlphabets` to get the vector of alphabets.
   - Calls `printAlphabets` to print the alphabets.

This code will output the lowercase and uppercase letters in alphabetical order.
