```cpp
#include <iostream>
#include <cctype>

/**
 * count the number of numbers in a string
 *
 * @param {std::string} str - The string from which to count numeric digits.
 * @returns {int} - The total count of numeric digits found in the string.
 */
int countNumbers(const std::string& str) {
    int count = 0;
    for (char c : str) {
        if (isdigit(c)) {
            count++;
        }
    }
    return count;
}

int main() {
    // Test the function
    std::string test_str = "Hello123World456";
    std::cout << "Number of digits: " << countNumbers(test_str) << std::endl;  // Outputs: Number of digits: 6

    return 0;
}
```

This C++ code defines a function `countNumbers` that iterates over each character in the input string and increments a counter if the character is a digit. It uses the standard library function `isdigit` to check whether a character is a digit.

The `main` function demonstrates how to use the `countNumbers` function with a sample string.
