```typescript
function compressWhitespace(inputString: string): string {
    // Split the input string by whitespace and join with a single space
    return inputString.split(/\s+/).join(' ');
}
```

### Explanation:
1. **Function Definition**: 
   - The function `compressWhitespace` takes a parameter `inputString` of type `string` and returns a `string`.

2. **Splitting the String**:
   - `inputString.split(/\s+/)` splits the input string using a regular expression `/\s+/`, which matches one or more whitespace characters.

3. **Joining the Parts**:
   - `.join(' ')` joins the resulting array of strings with a single space.

This TypeScript function replicates the functionality of the Python function, ensuring that multiple consecutive whitespace characters are compressed into a single space.
