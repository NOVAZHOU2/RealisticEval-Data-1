```typescript
/**
 * Compress multiple consecutive whitespace characters in a string into a single space.
 *
 * @param inputString - The string to be processed.
 * @returns The processed string with compressed whitespace.
 */
function compressWhitespace(inputString: string): string {
    return inputString.replace(/\s+/g, ' ');
}
```

### Explanation:
- **Function Signature**: The `compressWhitespace` function takes a single parameter `inputString` of type `string` and returns a `string`.
- **Regex Replacement**: The `replace` method uses a regular expression `\s+` which matches one or more whitespace characters (`\s`) globally (`g`). It replaces these matched sequences with a single space `' '`.

This TypeScript function will achieve the same result as the original Python function, compressing multiple consecutive whitespace characters into a single space.
