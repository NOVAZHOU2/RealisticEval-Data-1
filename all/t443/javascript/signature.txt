Sure! Here is the JavaScript version of your Python function, keeping the same comments:

```javascript
/**
 * Compress multiple consecutive whitespace characters in a string into a single space.
 *
 * @param {string} inputString - The string to be processed.
 * @returns {string} The processed string with compressed whitespace.
 */
function compressWhitespace(inputString) {
    return inputString.replace(/\s+/g, ' ');
}
```

In this JavaScript function:
- `/** ... */` is used for multi-line comments.
- `@param` and `@returns` tags are used to document the parameters and return value, similar to Python's docstrings.
- `inputString` is the parameter name.
- `replace(/\s+/g, ' ')` is used to replace one or more whitespace characters (`\s+`) globally (`g`) with a single space.
