Here is the equivalent Java code:

```java
import java.util.ArrayList;
import java.util.List;

public class Answer {

    public static List<List<Integer>> divideList(List<Integer> lst, int n) {
        List<List<Integer>> result = new ArrayList<>();
        
        if (lst == null || lst.isEmpty()) {
            return result;
        }

        int size = lst.size();
        int batchSize = (size + n - 1) / n; // Round up division

        for (int i = 0; i < size; i += batchSize) {
            int end = Math.min(i + batchSize, size);
            result.add(lst.subList(i, end));
        }
        
        return result;
    }
}
```

Please note that in Java, we need to specify the type of elements in the `divideList` method, so I used `Integer`. Also, Java does not have built-in support for dynamic typing like Python, so we need to ensure that the input list is not null or empty before proceeding with the division.
