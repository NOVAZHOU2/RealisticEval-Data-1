Here is your JavaScript function converted into C++:

```cpp
#include <iostream>
using namespace std;

/**
 * Calculates the discount percentage based on the given price and the actual price paid.
 *
 * @param originalPrice - The original price of the item.
 * @param actualPrice - The actual price paid for the item.
 * @return - The discount percentage, rounded to two decimal places.
 */
double calculateDiscount(double originalPrice, double actualPrice)
{
    if (originalPrice == 0 || actualPrice > originalPrice)
        return 0;

    double discount = ((originalPrice - actualPrice) / originalPrice) * 100;
    return round(discount * 100) / 100; // Rounding to two decimal places
}

int main() {
    double originalPrice = 100;
    double actualPrice = 80;
    cout << "The discount is: " << calculateDiscount(originalPrice, actualPrice) << "%" << endl;
    return 0;
}
```
Please note that you need to include the `<cmath>` library in order to use the `round` function in C++. Also, remember that C++ uses `::` instead of `.` to access methods from classes or namespaces.

This code will take an original price and an actual price, calculate the discount, and then print it out. If the actual price is greater than the original price or the original price is zero, the function returns 0.
