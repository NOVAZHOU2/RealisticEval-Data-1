```java
/**
 * Calculates the discount percentage based on the given price and the actual price paid.
 *
 * @param originalPrice - The original price of the item.
 * @param actualPrice - The actual price paid for the item.
 * @returns - The discount percentage, rounded to two decimal places.
 * @throws IllegalArgumentException if originalPrice is less than or equal to zero or if actualPrice is negative.
 */
public class DiscountCalculator {
    
    public static double calculateDiscount(double originalPrice, double actualPrice) {
        // Validate input
        if (originalPrice <= 0) {
            throw new IllegalArgumentException("Original price must be greater than zero.");
        }
        if (actualPrice < 0) {
            throw new IllegalArgumentException("Actual price cannot be negative.");
        }

        // Calculate the discount
        double discountAmount = originalPrice - actualPrice;
        double discountPercentage = (discountAmount / originalPrice) * 100;

        // Return the discount percentage, rounded to two decimal places
        return Math.round(discountPercentage * 100.0) / 100.0;
    }
}
```

1. **Method Signature**: Changed from JavaScript's function declaration to a static method in a Java class.
2. **Input Validation**: Used `IllegalArgumentException` for error handling in Java.
