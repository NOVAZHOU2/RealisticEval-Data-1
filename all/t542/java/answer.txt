```java
/**
 * Calculates the discount percentage based on the given price and the actual price paid.
 *
 * @param originalPrice - The original price of the item.
 * @param actualPrice - The actual price paid for the item.
 * @return The discount percentage, rounded to two decimal places.
 */
public double calculateDiscount(double originalPrice, double actualPrice) {
    // Calculate the difference between the original price and the actual price
    double difference = originalPrice - actualPrice;

    // Calculate the discount rate
    double discountRate = (difference / originalPrice) * 100;

    // Round off to two decimal places
    return Math.round(discountRate * 100.0) / 100.0;
}
```

This method calculates the discount rate as a percentage by subtracting the actual price from the original price, dividing the result by the original price, and multiplying by 100. It then rounds the result to two decimal places using the `Math.round` method.
