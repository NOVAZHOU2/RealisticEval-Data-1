Sure! Here is the equivalent JavaScript function for flattening a nested list:

```javascript
function flatten(nestedList) {
    /**
     * Flattens a nested list into a single list.
     *
     * @param {Array} nestedList - A list that may contain nested lists.
     * @returns {Array} A flattened version of the input list.
     */
    
    let result = [];

    function flattenHelper(subList) {
        subList.forEach(item => {
            if (Array.isArray(item)) {
                flattenHelper(item);
            } else {
                result.push(item);
            }
        });
    }

    flattenHelper(nestedList);
    return result;
}
```

This JavaScript function `flatten` takes an array `nestedList` as an argument and returns a flattened version of it. It uses a helper function `flattenHelper` to recursively traverse through the nested arrays and push non-array elements into the `result` array.
