```javascript
function probabilityOfRedBalls(n, x, y) {
    // calculate combinations
    const combination = (n, k) => {
        let result = 1;
        for(let i = 0; i < k; i++) {
            result *= (n - i);
            result /= (i + 1);
        }
        return result;
    }

    // calculate probability
    const totalBalls = x + y;
    const numerator = combination(x, n) * combination(y, totalBalls - n);
    const denominator = combination(totalBalls, n);

    return numerator / denominator;
}
```

This JavaScript function uses a helper function `combination` to calculate the number of ways to choose `k` items from `n` items without regard to order, which is essential for calculating the probabilities involved. Then it calculates the probability of drawing `n` red balls from a total of `x+y` balls using these combinations.
