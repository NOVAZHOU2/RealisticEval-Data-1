```typescript
function convertScoreToDecimal(scoreStr: string): number | null {
    /**
     * Converts the string representation of a score to its decimal value.
     * For example:
     *      input: "10/4"
     *      output: 2.5
     * Args:
     *      scoreStr (string): The score as a string, can be a decimal or a fraction
     *
     * Returns:
     *      number: The decimal value of the score as a number, or null if the input is invalid
     */

    const parts = scoreStr.split('/');
    if (parts.length !== 2) return null;

    const numerator = parseFloat(parts[0]);
    const denominator = parseFloat(parts[1]);

    if (isNaN(numerator) || isNaN(denominator)) return null;
    if (denominator === 0) return null;

    return numerator / denominator;
}
```

This TypeScript function does the same thing as your Python function. It splits the input string on the `/` character, checks that there are exactly two parts, and then parses those parts into floating-point numbers. If any part cannot be parsed into a number or if the denominator is zero, it returns `null`. Otherwise, it divides the numerator by the denominator and returns the result.
