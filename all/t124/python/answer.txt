To convert the given JavaScript shuffle function into Python, we will follow the same logic and adapt it to Python's syntax and conventions. Here's the equivalent Python code:
JavaScript Code
javascript
function shuffle(array) {
    let currentIndex = array.length;
    while (currentIndex > 0) {
        const randomIndex = Math.floor(Math.random() * currentIndex);
        currentIndex--;
        // Swap the elements at currentIndex and randomIndex
        const temp = array[currentIndex];
        array[currentIndex] = array[randomIndex];
        array[randomIndex] = temp;
    }
    return array;
}
Equivalent Python Code
python
import random

def shuffle(array):
    current_index = len(array)
    while current_index > 0:
        random_index = random.randint(0, current_index - 1)  # Get a random index
        current_index -= 1
        # Swap the elements at current_index and random_index
        array[current_index], array[random_index] = array[random_index], array[current_index]
    return array
Explanation of the Python Code:
Importing the Random Module: We import the random module, which provides functions to generate random numbers.
Function Definition: The shuffle function is defined using the def keyword.
Getting the Length of the Array: current_index is set to the length of the input list using len(array).
While Loop: A while loop is used to iterate until current_index is greater than 0.
Generating a Random Index: Instead of using Math.floor(Math.random() * currentIndex) as in JavaScript, we use random.randint(0, current_index - 1) in Python to get a random index within the appropriate range (inclusive of both ends).
Swapping Elements: The elements at current_index and random_index are swapped using tuple unpacking: array[current_index], array[random_index] = array[random_index], array[current_index].
Returning the Shuffled Array: Finally, the shuffled array is returned.
Usage Example
Here's how you can use the shuffle function:
python
# Example usage
my_list = [1, 2, 3, 4, 5]
shuffled_list = shuffle(my_list)
print(shuffled_list)  # Output will be a shuffled version of my_list
This code will give you a randomly shuffled version of the original list.
