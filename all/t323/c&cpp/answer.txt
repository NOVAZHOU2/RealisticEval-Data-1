Certainly! Below is the equivalent C++ code for your JavaScript function `isValidUsername`. This C++ function checks if a given string is a valid username containing only letters, numbers, and underscores.

```cpp
#include <iostream>
#include <regex>
#include <string>

/**
 * Verify that a string is a valid username and check that the username contains only letters, numbers, and underscores
 *
 * @param username - The username to be validated.
 * @return true if the username matches the regex pattern, indicating it is valid; otherwise, return false.
 */
bool isValidUsername(const std::string& username) {
    // Regular expression to match the allowed characters in a username
    std::regex usernameRegex(R"(^[a-zA-Z0-9_]+$)");
    
    // Check if the username matches the regular expression
    return std::regex_match(username, usernameRegex);
}

int main() {
    // Test cases
    std::string usernames[] = {"JohnDoe123", "Jane_Doe", "Invalid@User", "Valid_User"};
    
    for (const auto& username : usernames) {
        if (isValidUsername(username)) {
            std::cout << "Username \"" << username << "\" is valid." << std::endl;
        } else {
            std::cout << "Username \"" << username << "\" is invalid." << std::endl;
        }
    }

    return 0;
}
```

### Explanation:
1. **Regular Expression**: The regular expression `R"(^[a-zA-Z0-9_]+$)"` is used to match strings that consist solely of letters (`a-z`, `A-Z`), digits (`0-9`), and underscores (`_`). The `^` and `$` anchors ensure that the entire string must match the pattern.
2. **std::regex**: The `std::regex` class from the `<regex>` header is used to compile and apply the regular expression.
3. **std::regex_match**: The `std::regex_match` function checks if the entire input string matches the regular expression.
