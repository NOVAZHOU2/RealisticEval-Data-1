Sure, here is the equivalent Java method with the same functionality and comments:

```java
/**
 * Verify that a string is a valid username and check that the username contains only letters, numbers, and underscores.
 *
 * @param username - The username to be validated.
 * @return boolean - Returns true if the username matches the regex pattern, indicating it is valid;
 *                  returns false if the username contains any characters outside of letters, numbers, and underscores.
 */
public static boolean isValidUsername(String username) {
    // Regex pattern for validating the username
    String regex = "^[a-zA-Z0-9_]+$";
    
    // Check if the username matches the regex pattern
    return username.matches(regex);
}
```

In this Java code:
- We use `String.matches()` method to check if the input string `username` matches the specified regular expression.
- The regular expression `^[a-zA-Z0-9_]+$` ensures that the username consists only of letters (both uppercase and lowercase), digits, and underscores, and does not allow any other characters.
- The method returns `true` if the username is valid according to the regex, otherwise it returns `false`.
