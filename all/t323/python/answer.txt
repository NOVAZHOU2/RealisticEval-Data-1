Here's how you can translate your JavaScript function into Python:

```python
import re

def is_valid_username(username):
    """
    Verify that a string is a valid username and check that the username contains only letters, numbers, and underscores.

    :param str username: The username to be validated.
    :return: True if the username matches the regex pattern, False otherwise.
    :rtype: bool
    """

    # Define the regex pattern for a valid username
    pattern = r'^[a-zA-Z0-9_]+$'

    # Check if the username matches the pattern
    return bool(re.match(pattern, username))
```

In this Python version:
- We use the `re` module to handle regular expressions.
- The `is_valid_username` function takes a single parameter `username`, which is expected to be a string.
- Inside the function, we define a regular expression pattern that matches strings containing only letters (both uppercase and lowercase), digits, and underscores.
- We then use `re.match()` to see if the entire `username` string conforms to this pattern.
- If it does, `re.match()` returns a match object, which is truthy when converted to a boolean; otherwise, it returns `None`, which is falsy.
- Therefore, `bool(re.match(pattern, username))` will return `True` if the username is valid according to our criteria, and `False` otherwise.
