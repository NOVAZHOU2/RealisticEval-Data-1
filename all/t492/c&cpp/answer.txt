```cpp
#include <iostream>
#include <fstream>
#include <sstream>
#include <string>

void save_content_to_file(const std::string& content, const std::string& path) {
    /**
     * Saves the provided content to a specified file after cleaning up
     * redundant whitespace.
     *
     * @param content The text content to be saved to the file.
     * @param path The file path where the content will be saved.
     */

    // Remove redundant whitespace
    std::stringstream ss(content);
    std::string line;
    std::string cleanedContent;
    bool firstLine = true;

    while (std::getline(ss, line)) {
        if (!line.empty()) {
            if (!firstLine) {
                cleanedContent += "\n";
            }
            cleanedContent += line;
            firstLine = false;
        }
    }

    // Write the cleaned content to the file
    std::ofstream outFile(path);
    if (outFile.is_open()) {
        outFile << cleanedContent;
        outFile.close();
    } else {
        std::cerr << "Unable to open file: " << path << std::endl;
    }
}

int main() {
    std::string content = "  Hello   World \n\nThis is a test.\n\n\nAnother line.";
    std::string path = "output.txt";

    save_content_to_file(content, path);

    return 0;
}
```

### Explanation:
1. **Whitespace Cleanup**:
   - We use a `std::stringstream` to read through each line of the input content.
   - We concatenate non-empty lines with a newline character (`\n`) in between, ensuring that there is only one newline between lines and no leading/trailing spaces.

2. **File Writing**:
   - We use `std::ofstream` to write the cleaned content to the specified file path.
   - If the file cannot be opened, an error message is printed to the standard error stream.

3. **Main Function**:
   - A simple example is provided in the `main` function to demonstrate how to call `save_content_to_file`.
